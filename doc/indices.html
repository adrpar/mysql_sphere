7. Create an index

Indices are not yet supported by mysql_sphere, since MySQL does not allow easy extension of the indexing facility. The developer of mysql_sphere still needs to fully understand how pgSphere does the indexing (it is not really documented) and whether this can be implemented easily by a smart hashing algorithm that then uses the MySQL b-tree index provided with most storage engines.
